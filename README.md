##Online Retail Sales Analytics
End-to-End Data Analytics Project

##Project Overview

This project is a real-world Retail Sales Analytics solution that demonstrates a complete end-to-end data analytics workflow, from raw data cleaning to SQL analysis and business intelligence reporting using Power BI.

##Technologies used:
Python (Pandas, NumPy)
PostgreSQL for analytical SQL
Power BI for dashboard and insights
Git and GitHub for version control

##What This Project Includes
Data cleaning with 15+ real-world steps
15 professional-level SQL analysis queries
Exploratory Data Analysis (EDA)
Interactive Power BI dashboard
Customer, product, and revenue insights

##Tech Stack
-Python – Pandas, NumPy
-PostgreSQL – SQL, CTEs, Window Functions
-Power BI – Dashboard, DAX
-Tools – Jupyter Notebook, Git, GitHub
-Data Cleaning Workflow
-Notebook: data_cleaning.ipynb

##Key data cleaning steps performed:
-Removed duplicate records
-Handled missing values
-Standardized data types
-Cleaned text fields (trim, lowercase, remove special characters)
-Removed negative and invalid quantities
-Standardized country names
-Created TotalPrice column
-Cleaned product descriptions
-Converted multiple columns to numeric
-Removed outliers using the IQR method
-Renamed columns using snake_case
-Filtered invalid customer IDs

##SQL Analysis (PostgreSQL)
-The cleaned dataset was imported into PostgreSQL for analytical queries.
-Key SQL Analysis Performed
-Total revenue calculation
-Top selling products
-Revenue by country using ranking
-Monthly revenue trend
-Top 10 customers by revenue
-Category-wise revenue contribution (%)
-Orders by day of the week
-Revenue by sales channel
-Average order value (AOV)
-High-value orders
-Monthly customer growth
-Top revenue cities
-Payment method share using window functions
-Customer repeat purchase analysis
-Top 5 categories per country using CTEs
-SQL File: analysis_queries.sql

##Key Insights
-The United Kingdom generates the highest revenue
-A small number of products contribute most of the total revenue
-High-value customers drive a significant share of sales
-Credit card and online payments dominate transactions
-Clear seasonal trends are visible in revenue data
